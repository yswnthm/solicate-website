## 5. Technical Implementation Guidelines
- **Directory Routing**: Create a new subfolder under the root for each blog named using its slug, e.g., `/custom-website-vs-wordpress/index.html`.
- **Styling**: Enforce classes matching `tokens.css` values:
  - Font Sans for paragraphs and content.
  - Font Mono for code blocks, keywords, and technical figures.
  - Standard container layout (`class="container"`) with padding.
- **Markups**: Wrap content in semantic `<article>` tags. Each heading must follow strict order: `h1` (title) -> `h2` -> `h3`.
- **Metadata**: Embed structured LocalBusiness/FAQ/Service schema scripts in the header.
