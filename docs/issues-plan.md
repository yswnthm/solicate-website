# Website Revamp - Issues Plan

This document tracks the issues and improvements planned for the Solicate website, as discussed.

## 1. Hero Section Padding
- **Issue**: There is unnecessary padding at the start of the hero section across all pages.
- **Goal**: Reduce the top padding so the layout matches the desired reference (where the content sits higher up, similar to the second image).

## 2. Empty Bottom Section
- **Issue**: There is a mysterious empty section appearing at the bottom of the pages (likely just before the footer).
- **Goal**: Locate the empty `div` or `section` in the HTML/CSS and remove it to clean up the page flow.

## 3. Strong CTA with WhatsApp
- **Issue**: The bottom of the page lacks a strong final Call to Action (CTA).
- **Goal**: Add a dedicated, high-impact CTA section at the bottom of the pages featuring a prominent "WhatsApp" button to drive conversions.

## 4. FAQ Interactions
- **Issue**: The hover and click effects on the FAQ questions feel laggy and unnecessary, though the opened question view itself looks good.
- **Goal**: Remove the laggy hover/click animations on the FAQ questions to improve performance and feel, while preserving the design of the open state.

## 5. Highlight Alignment (Orange Border/Number)
- **Issue**: On hover, the orange highlight on the left side of the number seems misaligned or broken; it should be closer to the border.
- **Goal**: Adjust the CSS positioning (padding/margins/left property) of the orange highlight indicator so it sits flush or appropriately close to the border.

## 6. Hub Section Cards
- **Issue**: The Hub section currently uses square cards, which isn't the desired layout.
- **Goal**: Redesign the Hub cards to be wide, horizontal ("long") cards stacked one by one, allowing enough space for longer, more detailed descriptions.

## 7. "Stop guessing what's wrong." Section
- **Issue**: The section with the heading "Stop guessing what's wrong." lacks sufficient context.
- **Goal**: Add more descriptive text/copy to this section to better explain the value proposition or process.
